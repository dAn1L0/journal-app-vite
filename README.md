# JournalApp con Vite


### Configurar `variables`-> renombrar archivo `default-env` a `.env`


#### Para las imágenes -> [Cloudinary](https://cloudinary.com)
#### Para las notas -> [Firebase](https://firebase.google.com/)

#
Install:
```
$ yarn install
```
Run:
```
$ yarn dev
```
